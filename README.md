
# WEB POTOFOLIO

MY PORTOFOLIO WITH NEXT-JS


## Tech Stack

React, Next-Js, Vercel


## Cloning Project

Clone my-project with git clone

```bash
  git clone https://github.com/mrbayss/portofolio-new.git
  cd portofolio-new
```
    
## Running

To Running this project run

```bash
  npm run dev
```


## Running Tests

To run tests, run in browser with url

```bash
  http://localhost:3000
```
